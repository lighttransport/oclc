# OCLC: Simple offline OpenCL compiler using OpenCL API.

Maybe good to test each vendor's OpenCL compilation. 

## Requirement

* OpenCL SDK from each vendor
* premake4
* C++ compiler

## Build

    $ premake4 gmake
    $ make

## Usage

    $ oclcc <options> input.cl
